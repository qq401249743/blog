网络互联技术及应用实践-现代互联网概述

发表于 2018-09-15

1. 现代互联网结构

  • 以交换机为核心的现代局域网技术
  • 通过路由器与公共网络相连
  • 通过访问服务器使远程、分散用户入网


2. 公用网络

这些公用网络有些是在与Internet的竞争中被淘汰了的，然而被淘汰并不能说明它们不如Internet

  • PSTN
  • ISDN
  • DDN(国内曾建过骨干网)
  • X.25：第一种大规模使用的数据网络，在70、80年代有非常好的发展
  • ChinaPAC：当初电信公司经营的拨号入网的那种网络
  • Frame Relay：帧中继，是在X.25网络基础上发展的新一代网络技术
  • SDH，Synchronous Digital Hierarchy
  • DWDM，Dense Wavelegnth Division Multiplexing，密集波分复用
  • CSD电路交换数据业务
  • SMS短消息业务
  • GPRS+3G+4G


3. 局域网网络

3.1 传统局域网技术

包括Ethernet(10 base 5,2,T)，Token Ring，Arcnet，Plannet等

特点：任何时候最多只允许两个站点之间进行通信

提升带宽的办法：用路由器、网桥进行分段

几个易混淆的概念：

  • Ethernet、Token Ring、FDDI、Arcnet等都是物理网络,只具有低二层的功能,安装什么网络操作系统(NOS)就是什么网
  • 同一网段上可以同时存在不同的NOS，但不同协议之间的点不能进行通信。
  • Novell开发了OPT，使得可有多种协议并存。
  • 同一网段上不允许不同类型的网卡接入(Token Ring, Ethernet)
  • 路由器运行在网络层

3.2 现代局域网技术

现代局域网的核心是交换机，其每个接口可以接入一个网段

  1. 交换机分类：

  • 全部为10mbps的固定端口的低档Switch, 一般采用Cut-through，仍存在冲突。不适合连接服务器。例如：3COM的Link Switch 500 具有10个10 base T端口

  • 具有高速端口、多个10mbps端口的低档Switch, 采用Store and forward，冲突较少。高速端口可以是100 base T、100 base VG any LAN、FDDI、CDDI等。适合连接服务器、连接骨干网等

  • 具有灵活模块配置的中档Switch。可选用不同类型的模块，如10mbps、100mbps、155mATM模块等。例如：3COM的Link Switch 100具有12/24个10 mbps端口，1个100mbps端口，选加一个不同介质的Downlink模块用于连入Ethernet主干。

  • 具有多个100mbps、155mbsp甚至1000mbps的主干交换机，用于楼层间高速交换。例如：3COM的Link Switch 3000 具有8个100 base TX端口，5个100 base FX端口，可选配高速Downlink模块(100 base T、100 base F、155m ATM等)。

    三层交换机有路由功能

  1. VLAN

    VLAN定义：一组与位置和拓扑结构无关的设备,他们之间可以像在同一物理LAN一样通信。这意味着LAN段不受物理连接的硬件限制.而可以按使用的软件灵活分组. 例如: 按组织结构分组, 按应用分组。

    VLAN在逻辑上等价于广播域，在同一VLAN中，站点间可以任意通信

    VLAN的优点：

    • 在IP网上设备可以改变或移动
    • 有助于控制广播流量（限定在同一VLAN内）
    • 提供附加的安全性。VLAN间的站点通信，要经过路由器
    • 隔离作用。一个VLAN没有和路由器相连，此VLAN被隔离

    VLAN的种类：

    • 基于端口的VLAN：一个端口支持并只属于一个VLAN
    • 基于MAC的VLAN：按网卡地址划分，一个VLAN有一个MAC表
    • 基于第三层的VLAN：按IP地址分配
    • 基于策略的VLAN：不同的VLAN可选用上述三种不同的方法


4. 接入网

  • xDSL，如ADSL，SDSL，VDSL等
  • HFC，Hybrid Fiber Coax，光纤同轴混合
  • FTTx：
    • FTTH (Fiber To Home) 光纤到户
    • FTTB (Fiber To Building) 光纤到大楼
    • FTTC (Fiber To Curb) 光纤到路边
    • FTTO (Fiber To Office) 光纤到办公室

5. 家庭网

家庭网络（Home Network）指的是融合家庭控制网络和多媒体信息网络于一体的家庭信息化平台，是在家庭范围内实现信息设备、通信设备、娱乐设备、家用电器、自动化设备、照明设备、保安（监控）装置及水电气热表设备、家庭求助报警等设备互联和管理，以及数据和多媒体信息共享的系统。
家庭网络系统构成了智能化家庭设备系统，提高了家庭生活、学习、工作、娱乐的品质，是数字化家庭的发展方向。

现在家庭网络一般都是光纤到户，然后在家里直接用路由器建立无线局域网
