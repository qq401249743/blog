分布式系统-绪论

发表于 2018-09-15

1. 概念

一个分布式系统是若干个独立的计算机的集合，但是对该系统的用户来说，感觉该系统就像一台计算机一样。

  • 硬件方面：每台计算机都是独立、自主的计算机
  • 软件方面：用户感觉在独占系统

注：要分清分布式系统和分布式计算机系统的区别，比如社会和单位都属于分布式系统，概念里的定义是狭义的定义，实际上单指了分布式计算机系统

1.1 从集中式系统到分布式系统

导致这个过程的原因主要有两个：

  • PC（高性能微型计算机）的普及
  • 高速计算机网络的普及

当现有的系统拥有大量并发量时，一个解决方案是增加服务器，建立网站的副本从而分流，这种办法类似于网游的副本中人多时，可以切换线路缓解，每个线路的副本都是一样的。但增加服务器产生网站副本同时会引发一致性问题、容错问题和通信问题等多种问题。

1.2 分布式系统的应用

银行“一卡通”系统，连锁店“供应链”系统，传感器网络系统，企业“工作流”系统，万维网

1.3 一些相似的其它系统

P2P系统，云计算，普适计算

1.4 和并行计算的比较

并行计算（Parallel Computing）是指同时使用多种计算资源解决计算问题的过程。为执行并行计算，计算资源应包括一台配有多处理机（并行处理）的计算机、一个与网络相连的计算机专有编号，或者两者结合使用。并行计算的主要目的是快速解决大型且复杂的计算问题。

分布式计算是一门计算机科学，它研究如何把一个需要非常巨大的计算能力才能解决的问题分成许多小的部分，然后把这些部分分配给许多计算机进行处理，最后把这些计算结果综合起来得到最终的结果

一个更详细的对比参见：科普：并行计算、分布式计算、集群计算和云计算


2. 分布式系统的特点

2.1 与集中式系统比较

  1. 经济性：高的性价比

    因为当机器性能增加（一般用功耗增加代指），价格则会以平方的形式增加，所以性能越高增加性能的性价比就越低。举例说明，使用一台功耗为100的服务器不如使用两台功耗50的服务器

  2. 性能：

    集中式系统的性能是不可能无限增加的，因为CPU的温度会越来越高

  3. 应用固有的分布性：

    如支持协同工作的计算机任务，工作流等

  4. 可靠性高：

    多工系统具有较强的容错能力，如大公司的服务器集群每时每刻都有服务器在出问题，但整体的稳定运行仍可以保证

  5. 可扩充性强：

    指的是系统的演进能力，分布式系统使用多个服务器，增加服务器的数量是一件较为容易的事情

2.2 存在的问题

  1. 复杂性：系统和需求越来越复杂，软件和应用不断面临新的挑战
    • 操作系统、中间件、编程语言、工具
  2. 网络带宽：网络性能的限制
    • QoS（带宽、速度）
  3. 安全性和可信性：存在很大的安全隐患
    • 黑客、泄密、盗用、破环


3. 分布式系统的设计目标

3.1 透明性

  • 访问透明性：用户不必知道数据表示和访问方法
  • 位置透明性：用户不必知道资源位于何处
  • 迁移透明性：资源可以不改名，而随意移动
  • 重定位透明性：资源可在使用中移动位置
  • 复制透明性：用户不必知道有多少个备份存在
  • 并发透明性：多个用户可以自动地共享资源
  • 故障透明性：用户不必了解故障恢复处理

3.2 开放性

是一个开放式系统，符合标准协议都可以通过接口接入，其特点有

  • 互操作性：不同系统壳共同工作
  • 可移植性：应用程序可在不同系统上运行
  • 灵活性：可配置不同开发者的构件
  • 可扩展性：可增减构件

3.3 可伸缩性

主要包括规模可伸缩性、地理位置可伸缩性和管理可伸缩性三个方面

限制规模可伸缩性的问题包括：

  • 集中式服务（如高度机密的信息）
  • 集中式数据（通信线路影响）
  • 集中式算法（如最佳路由选择）

限制地理可伸缩性的问题包括：

  • 通信延迟
  • 不可靠通信
  • 集中式方案

限制管理可伸缩性的问题主要指策略冲突（收费、管理、安全性）

解决可伸缩性的主要技术有：

  • 减少通信延迟
    • 异步通信，减少等待
    • 增加客户端的工作量，减少服务器的工作量
  • 分布化
    • 如网络中的DNS服务
  • 复制和告诉缓存
    • 如Web cache

3.4 可靠性

  • 安全性：身份认证、入侵监测
  • 容错性：软硬件冗余，恢复和接管
  • 可用性：节点间“或”和“与”的相关性

3.5 高性能

  • 吞吐率：TPC-C基准测试
  • 实时性：响应速度
  • 并行性：细粒度并行性和粗粒度并行性


4. 分布式系统的类型

根据应用类型，主要分为三种：

  • 分布式计算系统：高性能科学计算
  • 分布式信息系统：信息管理和事务处理
  • 分布式普适系统：嵌入式应用
