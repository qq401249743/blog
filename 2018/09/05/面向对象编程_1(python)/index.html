面向对象编程_1(python)

发表于 2018-09-05 | 更新于 2018-09-06

面向对象的核心是程序是对象的集合，每个对象可以接收并处理其它对象的消息，程序的执行就是一系列消息在各个对象之间的传递。

面向过程的核心是程序是命令的集合，即一组函数的顺序执行，可以通过切分函数为小的子函数简化程序设计

1. 类(Class)和实例(Instance)

  • 类名首字母通常大写

  • (object)是父类名称

  • 实例创建通过类名调用完成，bart变量指向实例的内存地址

  • __init__前后分别有两个下划线，是特殊方法中的一种

  • 类中定义的函数第一个参数永远是self，表示创建的实例本身，调用时不需要传入该参数

  • 存在__init__方法时，创建实例必须传入与该方法匹配的参数

通过数据封装，我们可以隐藏类内的方法实现细节，使用时直接调用类及其方法即可。


2. 访问限制

将属性名name,score名称前加两个下划线，可以使其变为私有变量，阻止其被外部访问。

此时如果仍想访问类内属性，可以增加get_name,get_score,set_score这样的方法。

双下划线开头结尾的特殊变量可以直接访问，不受限制


3. 继承和多态

  • Animal继承了object类，Dog继承了Animal类
  • 继承可以使子类获得父类的全部功能
  • 子类Dog的run()方法覆盖了父类Animal的run()方法，覆盖就是一种多态
  • 多态是接口的多种不同实现方式

python作为动态语言，其继承体系不是严格的，参见鸭子类型


4. 获取对象信息

用于获取对象信息的主要有以下几个函数：

  • type()
  • isinstance()
  • dir()：返回一个包含判断对象所有属性和方法的list


5. 实例属性和类属性

通过实例变量或self变量给实例绑定属性

直接在类中定义的属性是类属性
