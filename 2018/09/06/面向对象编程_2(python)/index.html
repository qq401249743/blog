面向对象编程_2(python)

发表于 2018-09-06

除数据封装、继承和多态外，在python中，面向对象还有很多高级特性

1. __slots__

定义特殊变量__slots__可以限制该类实例能添加的属性

__slots__定义的属性仅对当前类实例起作用，对继承的子类不起作用


2. 多重继承

多重继承就是继承多个父类，这种设计称为Mixin，目的是给一个类增加多个功能

要和多级继承区别开来


3. 定制类

类似__slots__,__len__等，python提供了许多双下划线开头和结尾的函数帮助我们定制类，对这些方法的描述参见python的官方文档


4. 枚举类

python使用枚举类来更方便的定义和使用常量


5. 元类

依据元类可以创建类，依据类可以创建实例

但元类基本使用不到
