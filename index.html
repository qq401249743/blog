工程伦理-工程与伦理

发表于 2018-09-15

工程伦理问题涉及对工程行为正当性的思考和价值判断，往往需要在价值冲突中做出正确的价值选择。

1. 导论

1.1 工程伦理教育的意义

  • 有利于提升工程师的伦理素养，加强工程从业者的社会责任（最重要的一点）
  • 有利于推动可持续发展，实现人与自然的协同进化（倾向于环境方面）
  • 有利于协调社会各群体之间的利益关系，确保社会稳定和谐（强调要保证公众利益，不止为老板考虑）

1.2 工程伦理教育的目标

目标是提高工程师的伦理素质，其内涵包括：

  • 培养工程伦理意识和责任感（含义是增强对工程伦理问题的敏感性和责任感）
  • 掌握工程伦理的基本规范（指的是面对伦理问题时应遵循的行为准则）
  • 提高工程伦理的决策能力（面对复杂伦理问题时理性决策）

面向对象编程_1(python)

发表于 2018-09-05 | 更新于 2018-09-06

面向对象的核心是程序是对象的集合，每个对象可以接收并处理其它对象的消息，程序的执行就是一系列消息在各个对象之间的传递。

面向过程的核心是程序是命令的集合，即一组函数的顺序执行，可以通过切分函数为小的子函数简化程序设计

1. 类(Class)和实例(Instance)

  • 类名首字母通常大写

  • (object)是父类名称

  • 实例创建通过类名调用完成，bart变量指向实例的内存地址

  • __init__前后分别有两个下划线，是特殊方法中的一种

  • 类中定义的函数第一个参数永远是self，表示创建的实例本身，调用时不需要传入该参数

  • 存在__init__方法时，创建实例必须传入与该方法匹配的参数

通过数据封装，我们可以隐藏类内的方法实现细节，使用时直接调用类及其方法即可。

git学习-基础

发表于 2018-04-21 | 更新于 2018-05-13

仓库和文件

repository，中文名叫仓库，我的理解就是一个文件夹，里面放了同属一个项目的所有文件

创建一个仓库

  1. 在本地其实就是创建目录的命令,我在/home/shuzang目录下创建gittest文件夹

    $ mkdir gittest

  2. 进入创建的文件夹，使用 git init 命令把该目录变成git的仓库

    $ cd gittest
$ git init

    提示

    初始化空的Git仓库于/home/shuzang/gittest/.git/

  3. 然后仓库就建好了，该目录下会多一个.git文件夹，默认是隐藏的，用来跟踪管理仓库。使用ls -a命令可以看到该文件夹，默认隐藏说明它很重要，里面的文件改动的话很容易导致仓库损坏
